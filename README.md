# Douban_Book_System
豆瓣读书系统：实现书籍数据爬取与全生命周期管理的项目。


## 项目概述
本项目聚焦于模拟豆瓣读书的核心功能，包含两大核心模块：
- **书籍数据爬取模块**：定向获取书籍信息并完成标准化预处理，输出结构化数据源；
- **书籍管理模块**：提供书籍信息的增、删、改、查（CRUD）操作，配套权限管控逻辑。


## 核心功能
1. **数据爬取与预处理**
   - 抓取字段：书名、作者、ISBN、出版社、出版时间等；
   - 数据处理：去重、格式统一、字段校验，生成可直接入库的结构化数据。

2. **书籍管理（CRUD）**
   - 新增：支持单条书籍手动录入、爬虫数据批量导入；
   - 查询：基础ID/ISBN精准检索、高级关键词+多条件筛选（含分页）；
   - 编辑：基于唯一标识的信息更新，支持字段局部修改；
   - 删除：权限管控下的安全删除，避免误操作。
